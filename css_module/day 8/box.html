What Is the CSS Box Model?

The CSS Box Model defines how every HTML element is structured and spaced on a webpage. Each element is a rectangular box made up of the following layers:

  • Content: The innermost area where your text or images appear.
  • Padding: Space between the content and the border.
  • Border: The line that surrounds the padding.
  • Margin: The outermost space between the element and other elements.

The diagram above illustrates the hierarchy of the box model layers.

 div {
        width: 200px;
        height: 100px;
        padding: 10px;
        border: 5px solid black;
        margin: 20px;
      }

The actual space taken up by the element will be:

  • Content: 200px (width) x 100px (height)
  • Padding: 10px on all sides
  • Border: 5px on all sides
  • Margin: 20px on all sides

Total Width = Content Width + Padding + Border + Margin

Total Height = Content Height + Padding + Border + Margin

Detailed Properties

Content Area

  • Holds the actual content of the box
  • The size is determined by:
    width and height (default content size depends on the element type if unspecified).

Padding

  • Adds space around the content, inside the border.
  • Properties include:
    padding-top
    padding-right
    padding-bottom
    padding-left

Border

  • Surrounds the padding (or content if no padding is applied).
  • border-width (e.g., 2px)
  • border-style (e.g., solid, dashed, none)
  • border-color (e.g., #333, red)
  • border (shorthand)

Margin

  • Defines the space between the box and neighboring elements.
  • Properties include
    • margin-top
    • margin-bottom
    • margin-right
    • margin-left

Box Sizing in CSS

By default, an element's width and height only apply to the content, while padding, borders, and margins add extra space. You can adjust this behavior using the box-sizing property:

  1. Content-Box (Default)
    • width and height include only the content area.
    • Padding and border are added outside.
  2. Border-box
    • width and height include content, padding, and border.
    • This makes layout calculations easier.

/* Default behavior */
box-sizing: content-box;

/* Includes padding and border in width and height */
box-sizing: border-box;
