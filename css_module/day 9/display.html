Display

The display property is a very important CSS property for controlling layout. It specifiesif/ how an element is displayed. Many of the elements have default display property value as inline or block (inlineandblockelements in HTML).

The display property has following values:

  • block
  • inline
  • inline-block
  • none

Block Display

When display: block; is used these following happens to the element:

  • element starts in a new line
  • takes up full-width of the parent element

Example

originally the 2 spans look like this:

span1 span2

adding the block property to them like this:
span{ display: block; }
and it will be shown like this now:

span1 span2

Inline Display

When display: inline; is used these following happens to the element

  • element doesn't starts in a new line
  • only takes up as much width as necessary, so cannot set height andwidth
  • the vertical margins do not work

Example

originally a list looks like this:

  • one
  • two
  • three

making the list inline: li { display: inline; }
and the list will now be shown in a single line like this(the space before thelist isbecause ul element have a default padding value):

  • one
  • two
  • three

Inline-Block Display

display: inline-block; is a combination of the properties of the inline andblockelements. These are the advantages of inline-block:

  • height and width of the element can be set now
  • vertical margins are allowed
  • element can sit next to each other

Example

there are two empty div element:
and the following properties have been applied to them:

                .div {
                    display: inline-block;
                    border: 1px dashed black;
                    width: 100px;
                    height: 100px;
                    background-color: lightgrey;
                    
                    }

two divs will be displayed next to each other like this:

div1
div2

You can notice a space between these 2 divs. It is because inline elements haveaword-spacing in them and it is also a default in inline-block elements.

No Display

The display:none; property hides the elements in a browser. It actually removesthe element from the HTML page and nothing is shown in its place.

This is similar to another property - visibility:hidden;. The differenceis that the element is not removed from the page and still occupies the space.
