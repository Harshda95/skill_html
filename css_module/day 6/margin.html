What is Margin ?

The CSS margin properties are used to create space between the borders and theothersurrounding elements.
You can also provide negative margins as well.
There are two other values that are used for the margin: auto - the margin is applied by the browser itself only to horizontal margins
none - to remove any margins from the element or makes the value of margin equal tozero

Types of Margin

The margin property for the sides are:

  • Margin-left
  • Margin-right
  • Margin-top
  • Margin-bottom

What is Padding ?

The CSS padding properties are used to generate space around an element's content, andits borders.

types of Padding

  • Padding-left
  • Padding-Right
  • Padding-bottom
  • Padding-top
