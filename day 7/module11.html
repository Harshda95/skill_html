WHAT IS HTML?

● HTML stands for Hyper Text Markup Language.
● HTML describes the structure of web pages using markup.
● HTML is used to provide the content(words, images, audio, video, and so on) tothewebpages.
● HTML is a tag based language. They are defined within the angle brackets.
● HTML file can be created using a text editor.

COMMENTS IN HTML

The comment tag is used to insert comments in the source code. Comments arenotdisplayed in the browsers.
